What Is Return HTTP Status?

This site lets you pass over a HTTP status and then get that same code back as the response. It's useful for testing how your application would react to specific return codes.

How Do I Use It?

Generate a DELETE, GET, PATCH, POST, or PUT request to https://returnhttpstatus.com/code/{code}

Example:
GET https://returnhttpstatus.com/code/401

Output

All output is JSON formatted with a "success" (boolean) and "error" (string or null) fields. 
{
    success: true,
    error: null
}
            
{
    success: false,
    error: "Invalid code detected."
}

What Codes Are Valid?

Any 3-digit number is accepted.
